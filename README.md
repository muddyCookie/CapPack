# CapPack Website (Static)

This website now runs as plain static files.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/home.html`.

## Main files

- `home.html`
- `downloads.html`
- `version-compatibility.html`
- `credits.html`
- `styles.css`
- `script.js`

No build step or Node setup is required.
